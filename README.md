# Lovable Skills Toolkit 🚀

A curated collection of custom Lovable skills designed for building premium production-ready websites and webapps.

These skills help create projects that feel handcrafted by a high-end digital agency rather than AI-generated templates.

## Included Skills

### 🎨 Premium Agency Web Builder
Creates premium websites and webapps with strong UX, visual hierarchy and agency-level quality.

### 🔍 SEO Master
Adds SEO best practices including metadata, schema markup, semantic structure and search optimization.

### 🎯 Conversion Optimizer
Improves conversion rates with stronger CTA placement, trust elements and optimized user flows.

### ⚡ Performance Responsive Optimizer
Optimizes websites for:

- Mobile responsiveness
- Desktop responsiveness
- Fast loading
- Better Core Web Vitals

### 📊 SaaS Dashboard Builder
Builds organized dashboards, analytics systems and internal tools.

### 🔧 Backend Architect
Designs scalable backend systems including:

- APIs
- Authentication
- Database structure
- User roles
- Security

### 🎬 Animation Director
Creates subtle premium animations and micro interactions without hurting performance.

### 📐 PRD Architect
Creates detailed Product Requirement Documents before implementation.

Includes:

- User personas
- User journeys
- Requirements
- Technical planning
- Success metrics
